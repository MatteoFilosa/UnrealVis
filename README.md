# UnrealVis
The repository for UnrealVis, a diagnostic laboratory and optimization testbed for simulations regarding large 3D datasets, made from scratch in Unreal Engine.

# Starting a simulation

Select your desired optimizations from the "Select Optimization" menu
Press on "Start Simulation"
Load your desired dataset
Press E to spawn the 3D object
Explore the dataset with WASD and your mouse! Press P to start sampling performance data, and Q to quit and save the sampling.

# Analying performance data

Analyze performance data via the "Compare Charts" or "Analyze Performance". You can also download performance data as JSON in the last view.
